# Employee Attrition ML Dashboard

## 📌 Project Overview

This project is an End-to-End Machine Learning Dashboard built using:

* Decision Tree Classification
* Decision Tree Regression
* Streamlit
* Scikit-learn
* SMOTE
* Pandas
* Matplotlib

The application predicts:

1. Employee Attrition (Classification)
2. Employee Salary Prediction (Regression)

# ▶️ Run Project

## 1️⃣ Install Libraries

```bash
pip install -r requirements.txt
```

---

## 2️⃣ Data Preprocessing

```bash
python src/data_preprocessing.py
```

---

## 3️⃣ Train Classification Model

```bash
python src/train_model.py
```

---

## 4️⃣ Train Regression Model

```bash
python src/train_regression.py
```

---

## 5️⃣ Run Streamlit Dashboard

```bash
streamlit run app/streamlit_app.py
```
